About Pitch Profiler

Overview

The Pitch Profiler is an advanced baseball analytics tool designed to provide comprehensive analysis of pitching performance using Statcast data. It offers detailed insights into pitch movement, velocity, location patterns, and effectiveness metrics.

Key Features
  • Movement Analysis: Horizontal and vertical break analysis with visual profiles
  • Usage Patterns: Pitch frequency and performance metrics
  • Location Mapping: Strike zone heatmaps and location tendencies
  • Performance Metrics: CSW%, K%, BB%, and advanced statistics
  • Interactive Visualizations: Plotly-powered charts and graphs
  • PDF Reporting: Professional analysis reports
Data Requirements

The Pitch Profiler requires CSV files with Statcast pitch-by-pitch data. The following columns are essential for proper analysis:

Column Description Required
player_name Pitcher's name Yes
pitch_name Pitch type (FF, SL, CH, etc.) Yes
release_speed Pitch velocity in mph Yes
pfx_x, pfx_z Horizontal and vertical movement Yes
plate_x, plate_z Pitch location coordinates Yes
description Pitch outcome description Yes
events At-bat outcome events Optional
release_spin_rate Spin rate in RPM Optional
Analysis Outputs
Movement Profile
  • Horizontal vs. vertical break scatter plot
  • Pitch type clustering
  • Average movement by pitch type
  • Break magnitude circles
Usage & Performance
  • Pitch frequency analysis
  • Called Strike + Whiff (CSW) rates
  • Velocity and spin rate averages
  • Comparative pitch metrics
Technical Specifications
  • Framework: Dash (Plotly) for interactive visualizations
  • Data Processing: Pandas for data manipulation and analysis
  • Visualization: Plotly for charts and graphs
  • Reporting: ReportLab for PDF generation
  • File Support: CSV upload via drag-and-drop interface
Quick Start
  1. Click "Launch Pitch Profiler" to open the dashboard
  2. Upload your Statcast CSV file
  3. Review the automated analysis
  4. Explore different visualization tabs
  5. Generate and download PDF reports
Supported File Types
